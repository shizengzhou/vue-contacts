# vue-contacts
练习 Vue 和 Vue Router 的小 demo

## 运行 Server
- `git clone https://github.com/udacity/reactnd-contacts-server.git`
- `cd reactnd-contacts-server`
- `npm install`
- `node server.js`

## 运行项目
- `npm install`
- `npm run serve`

注：在 Terminal 中使用 git 提交带有反引号的信息时，反引号中的内容似乎被过滤了，在 VS Code 中提交则无此问题。
